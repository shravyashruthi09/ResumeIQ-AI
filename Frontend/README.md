# ResumeIQ AI

An AI-powered Resume Analyzer that evaluates resumes against job descriptions using ATS scoring and AI-based recommendations.

## Features

- ATS Resume Score
- Resume Summary
- Skills Analysis
- Resume Section Analysis
- Resume Quality Analysis
- Strengths & Weaknesses
- Category-wise Scores
- AI Recommendations
- Modern Responsive UI

---

## Tech Stack

### Frontend
- React
- Vite
- Axios

### Backend
- FastAPI
- SQLite
- SQLAlchemy
- spaCy
- PyMuPDF
