# ResumeIQ AI – AI-Powered Resume Analyzer

An AI-powered full-stack web application that analyzes resumes against job descriptions using Google Gemini AI, providing ATS scores, skills analysis, resume quality evaluation, and personalized improvement suggestions.

---

## 📖 Overview

ResumeIQ AI helps job seekers optimize their resumes by comparing them with specific job descriptions. The application extracts resume content, evaluates ATS compatibility, identifies missing skills, and generates AI-powered recommendations for improving resume quality.
